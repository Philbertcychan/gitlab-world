▶ Roadmap — why this becomes a live API

This prototype serves historical, static values. The build I'm working toward replaces every figure with a live fetch: a pipeline pulls each indicator from its source on a schedule (SEC EDGAR, FRED, GitHub Octoverse, GitLab's Innovation Graph, Gartner) and exposes it through the query layer above. The thesis: finance becomes an internal API — any team (product, GTM, leadership) calls for the revenue signal landscape on demand, instead of waiting on a deck. This is the same discipline as investor relations, turned inward and made programmatic.

Sources. GitLab metrics — SEC 8-K filings (FY2021–Q3 FY2026). Macro — Gartner press releases, FRED, BLS. Developer activity — GitHub Octoverse, GitLab Innovation Graph, SlashData. AI market — JetBrains AI Pulse, Stack Overflow Developer Survey, Microsoft earnings, analyst estimates. Sparse series shown with available points only; derived figures tagged DERIVED. Nothing fabricated — where data is thin, it's marked.
